# 开发历程与变更日志

本文件记录本仓库可核实的版本与开发里程碑。未创建 Git 提交或标签的工作统一标记为“未发布”，不追溯虚构版本号。

## 1.0.0 — 2026-07-13

基线：上游 `v0.1.3`（`27943c0`）。以下内容构成 Quota Beacon `1.0.0` 的首个独立开发版本。

### 新增

- 本机 Codex 历史 Token 汇总与会话 Token 浮层显示。
- 101 级额度颜色主题：五个可编辑关键色、连续插值、对比度处理，以及防止相邻整数百分比重色的自动测试。
- 独立色彩预览窗口与调色盘编辑窗口；支持 `0–100%` 滑块预览、恢复默认色和保存自定义色。

### 变更

- 主卡片和折叠悬浮球改用同一套额度主题算法；以 5 小时额度为主视觉，缺少 5 小时数据时回退显示周额度。
- 真实额度改为每 10 秒强制刷新；预览期间同步显示滑块所选额度，退出后恢复真实数据。
- 增强双窗口协作、移动跟随、可用工作区上下避让、高 DPI 物理尺寸处理，以及预览期间的置顶状态恢复。
- 为额度数据、设置加载、Token 格式化和中英文文案补齐类型、桥接和测试覆盖。
- 进度条末端使用无实体的白色柔光提示当前位置，避免圆形标记造成“可拖动按钮”的错误暗示。
- 前端、Rust 与 Tauri 构建版本统一提升为 `0.1.4`，用于区分包含上述功能的可恢复版本。

## 未版本化开发里程碑 — 2026-07-12

### 修复色卡预览透明边缘

- 移除透明色卡窗口边缘被裁切后形成灰色矩形的外部阴影。
- 完成前端测试、Vite 构建、Rust 检查和 Tauri 桌面构建；重建并替换本机安装程序进行验证。

### 修复设置错误提示残留

- 设置读取在启动阶段短暂失败时，错误横幅曾在后续额度正常更新后仍然保留。
- 增加设置读取重试，并在设置读取或桌面设置同步成功后清除过期错误状态。
